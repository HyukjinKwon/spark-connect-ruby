# Security Policy

## Supported versions

The latest released `0.x` minor version receives security fixes. As the project
is pre-1.0, please always test against the most recent release.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✅        |

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, report privately via GitHub's
[security advisory workflow](https://github.com/HyukjinKwon/spark-connect-ruby/security/advisories/new).

When reporting, please include:

- a description of the vulnerability and its impact,
- steps to reproduce or a proof of concept, and
- any suggested remediation, if known.

You can expect an initial acknowledgement within a few days. We will keep you
informed of progress and coordinate a disclosure timeline once a fix is ready.
